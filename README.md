## 🔧 How to Use  
To run any of these projects on your local machine:  
1. Clone the repository:  
   ```sh
   git clone https://github.com/Ridhsuki/flutter_lesson.git
   cd flutter_lesson
   ```
2. Navigate to a project folder and run:  
   ```sh
   flutter pub get
   flutter run
   ```
